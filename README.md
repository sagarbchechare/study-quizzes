IIT JEE Preparation Quiz Platform
Overview
A quiz platform for IIT JEE Main preparation, starting from Class 6 through Class 12. This repository provides progressive quizzes that build foundational concepts and advance toward competitive exam readiness.
Structure
Learning Phases
Class 6-8: Foundation

Core concepts in Physics, Chemistry, Mathematics
Basic problem-solving skills

Class 9-10: Pre-JEE

Advanced concepts
Application-based problems
Early JEE pattern introduction

Class 11-12: JEE Preparation

Complete JEE Main syllabus
Previous year papers
Mock tests

Features

Subject-wise quizzes (Physics, Chemistry, Mathematics)
Topic and chapter-wise organization
Timed practice tests
Detailed solutions
Progress tracking
